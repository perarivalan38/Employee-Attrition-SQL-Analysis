📊 Employee Attrition Analysis using SQL

⭐ Project Overview

* This project analyzes employee attrition data using SQL to uncover workforce trends and generate actionable business insights. The analysis focuses on employee turnover, salary distribution, departmental performance, and employee rankings using SQL analytical functions.

* The project demonstrates practical SQL skills commonly required for Data Analyst roles, including data exploration, aggregation, ranking, and business reporting.

## 📊 Objective:
Analyze employee data to understand:

* Employee attrition rate
* Department-wise employee distribution
* Salary analysis
* Highest-paid employees
* Workforce statistics

## 🛠 Tools Used:
* SQL (MySQL)
* MySQL Workbench
* CSV Dataset
* GitHub

## 📈 Business Analysis:
*Average Salary by Department
* Top Paid Employee in Each Department
* Top 10 Highest Paid Employees
